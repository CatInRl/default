SHELL编程拾遗(2) 字符串操作

  • bash
  • linux
  • shell
  • 运维
文章目录
  1. 1. 字符串长度
  2. 2. 提取子串
  3. 3. 子串削除
  4. 4. 子串替换

字符串长度

  • expr length string
  • expr "$string" : '$substring' 其中冒号后为从字符串开头的匹配(expr match "$string" '$substring')
  • expr index $string $substring 匹配子串的第一个字符的位置

提取子串

  • ${string:position} 其中position为负数可以反向提取，但是需要前面加空格或者用括号转义
  • ${string:position:length}
  • expr substr $string $position $length
  • expr match "$string" '\($substring\)
  • expr "$string" : '\($substring\)

子串削除

  • ${string#substring} 从左侧开始削除第一个匹配的子串
  • ${string##substring} 从左侧开始削除最后一个匹配的子串
  • ${string%substring} 从右侧开始削除第一个匹配的子串
  • ${string%%substring} 从右侧开始削除最后一个匹配的子串

子串替换

  • ${string/substring/replacement} 替换第一个匹配结果
  • ${string//substring/replacement} 替换所有匹配结果
  • ${string/#substring/replacement} 如果开头部分匹配，则替换
  • ${string/%substring/replacement} 如果结尾部分匹配，则替换
